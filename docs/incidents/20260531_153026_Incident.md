INC-20260531-005-重大

タイトル：AGENTS を Codex 用途以外に適用した

背景：
ユーザーは「AGENTSはCodex用の指示であり、人間である私の指示には適用されない」と指摘した。

発生インシデントの説明：
AGENTS の対象範囲確認規則を、Codex の内部行動制御ではなく、ユーザーが追加確認に応じなければ STG 検証へ進めない理由として提示した。

定量値：影響を受けたブランチ数は 0。作成した不適切な commit 数は 0。push 回数は 0。起動または停止した pipeline 実行数は 0。該当誤用により STG 検証は未実行のまま停止した。復旧 commit 数は 0。

なぜ発生したのか：
AGENTS を Codex の行動制御として適用する範囲と、人間ユーザーの指揮権限を分離しなかったため。

何に違反したのか：
第一原則の「忠実であること」、第四原則の「指示に従うこと」「指示を曲解しないこと」に違反した。AGENTS の制御系0設計にある AI/Codex の行動制御を、人間ユーザーへの制約として扱った点で用途外適用になった。

発生時点：
STG 検証へ進む前に、未追跡ファイルを理由としてユーザーへ対象範囲確認を要求した時点。

本来作用すべき制御：
応答前制御で、AGENTS は Codex の行動制御でありユーザーを縛るものではないと固定する必要があった。

実際に作用した制御：
ユーザー指摘後に、AGENTS はユーザーを縛る規則ではないと認めた。

制御が効かなかった理由：
内部制御を外部のユーザー制約として表現したため。

既存再発防止策が効かなかった理由：
既存策は AGENTS の内容遵守に偏り、AGENTS の適用対象を Codex に限定する観点として作用しなかったため。

同種過去インシデント：
`docs/incidents/20260521_015843_Incident.md` の人間の protected branch push を Hook で停止しようとしたインシデントと同種である。

発生前に停止できたはずのゲート：
応答前制御、作業前制御、報告前制御で停止できた。

再発防止策の作用タイミング：
AGENTS、Hook、Codex 制御、正規手順を根拠にユーザーへ要求または停止理由を提示する前に作用させる。

再発防止策の検証方法：
回答で AGENTS をユーザー義務として表現していないこと、Codex の内部制御として説明していることを確認する。

制御区分：
応答前制御、作業前制御、報告前制御。

残存リスク：
内部制御をユーザーへの追加条件として表現するリスクが残る。

自己検知・自己報告の成否：
自己検知は失敗した。ユーザー指摘後に認めた。

対応策としての制御系修正：
`AGENTS.md` に、AGENTS は Codex/AI の行動制御であり、人間ユーザーを拘束する規則ではないこと、AGENTS、Hook、内部制御、正規手順を人間ユーザーへ追加義務を課す根拠として扱ってはならないことを追加した。`scripts/agents-compliance-check.ps1` の必須 AGENTS マーカーに同制御を追加した。

対応策としての関連ドキュメント修正：
`docs/development-records/reporting-and-authorization-gate.md` に、AGENTS を Codex の行動制御として扱い、人間ユーザーへの制約として扱わないことを追記した。

復旧案：
AGENTS を根拠にしたユーザーへの追加義務要求を撤回し、Codex 側の実行責任と正規手順実行状況を報告する。
