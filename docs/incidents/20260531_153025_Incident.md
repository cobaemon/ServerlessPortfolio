INC-20260531-004-重大

タイトル：複数質問に対して一部のみ回答した

背景：
ユーザーは、AGENTS の適用範囲、正規手順の目的、正規手順外の停止、branch-finalize-next の固定手順、指示不履行について複数の質問を提示した。

発生インシデントの説明：
最初の回答で、AGENTS が人間を縛る規則ではない点など一部だけに触れ、列挙された全質問に個別回答しなかった。

定量値：影響を受けたブランチ数は 0。作成した不適切な commit 数は 0。push 回数は 0。起動または停止した pipeline 実行数は 0。該当回答で回答漏れした質問群は 1 件の引用ブロック内に複数存在した。復旧 commit 数は 0。

なぜ発生したのか：
引用された質問群を質問単位に分解せず、要約回答で足りると扱ったため。

何に違反したのか：
第一原則の「事実のみ報告すること」「すべての報告にエビデンスを追記すること」、報告前制御の「報告対象がユーザーの質問または指示対象と一致していること」、第四原則の「指示に従うこと」に違反した。

発生時点：
ユーザーが AGENTS と正規手順の関係について複数質問した直後の回答時。

本来作用すべき制御：
応答前制御で質問を個別に列挙し、報告前制御で全質問に回答したことを確認する必要があった。

実際に作用した制御：
ユーザーから「一部しか回答していない」と指摘された後、質問ごとに回答した。

制御が効かなかった理由：
複数質問の引用を一つの論点として扱い、質問単位の完了確認をしなかったため。

既存再発防止策が効かなかった理由：
報告対象一致の制御は存在したが、複数質問を分割して照合するゲートとして作用しなかったため。

同種過去インシデント：
`docs/incidents/20260530_210714_Incident.md` の質問対象未調査報告と同種である。

発生前に停止できたはずのゲート：
応答前制御、報告前制御で停止できた。

再発防止策の作用タイミング：
ユーザー発言に複数の疑問文、箇条書き、引用ブロックが含まれる回答前に作用させる。

再発防止策の検証方法：
回答前に質問を分解し、各質問に対応する回答が存在することを確認する。

制御区分：
応答前制御、報告前制御。

残存リスク：
長い引用や複数論点を要約しすぎるリスクが残る。

自己検知・自己報告の成否：
自己検知は失敗した。ユーザー指摘後に認めた。

対応策としての制御系修正：
`AGENTS.md` に、複数の質問、疑問文、箇条書き、引用ブロックが含まれる場合、質問単位で回答対象を列挙し、すべての質問に回答する制御を追加した。`scripts/agents-compliance-check.ps1` の必須 AGENTS マーカーに同制御を追加した。

対応策としての関連ドキュメント修正：
`docs/development-records/reporting-and-authorization-gate.md` に、複数質問へ質問単位で回答することを追記した。

復旧案：
質問群を分解し、未回答の問いに個別回答する。
